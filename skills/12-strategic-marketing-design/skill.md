# 12-strategic-marketing-design

## Objetivo

Alinear decisiones de marketing y diseno antes de producir UI para evitar retrabajo por estetica generica.

## Marco de trabajo

1. Definir audiencia, promesa y nivel de exclusividad
2. Aplicar filtro Luxury/Generic Gate (must-have y anti-patrones)
3. Elegir direccion de diseno con justificacion estrategica
4. Congelar reglas ejecutables: tokens, grid, componentes, tono de copy

## Criterios clave

- Coherencia entre posicionamiento y visual
- Diferenciacion frente a competidores
- Relevancia para audiencia premium
- Prioridad en conversion, no solo estetica
