# 00-shared

## Objetivo

Definir reglas transversales para cualquier tarea del website.

## Reglas

- Respetar el `BRIEF MAESTRO.md` como fuente de verdad.
- Priorizar conversion: aplicar a membresia.
- Mantener tono premium, estructurado y accesible.
- Evitar estetica wellness espiritual o diseno fitness generico.
- Direccion visual base: referencia estructural tipo Equinox, adaptada a la marca Tamara.
- Paleta obligatoria: negro profundo, gris grafito, blanco y acento rosa intenso.
- Motion: microanimaciones utiles y sutiles, nunca decorativas.
- Responsive real: experiencia impecable en desktop y telefonos.
