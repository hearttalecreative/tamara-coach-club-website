# 02-ui-visual

## Objetivo

Evaluar y construir un sistema visual maduro: tipografia, tokens, componentes y consistencia cross-screen.

## Framework de revision

- Sistema tipografico (escala, pesos, ritmo)
- Coherencia de diseno (espaciado, grid, tokens)
- Consistencia visual (botones, inputs, radios, sombras)
- Whitespace estrategico y composicion
- Color con rol funcional y contraste accesible
- Comportamiento responsive por breakpoint

## Salida esperada

- Clasificacion de madurez: improvisado, emergente, estructurado, maduro
- Lista de issues estructurales de alto impacto
- Acciones priorizadas para escalabilidad
