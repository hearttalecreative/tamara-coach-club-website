# 04-frontend-development

## Objetivo

Implementar y optimizar front-end con HTML semantico, CSS escalable, JS mantenible y performance estable.

## Checklist

- Estructura semantica y accesibilidad base
- Layout definido por responsabilidad (Flex/Grid)
- Media queries mobile-first por comportamiento real
- JS modular (DOM, eventos, logica separados)
- Lazy loading e imagenes optimizadas sin romper LCP
- Integraciones externas no bloqueantes
