# 05-css-animations

## Objetivo

Disenar sistema de animacion CSS orientado a UX: claro, sobrio y de alto rendimiento.

## Principios

- Toda animacion tiene proposito (feedback, foco, jerarquia)
- Duraciones orientativas: 150ms a 600ms
- Easing natural (ease-out/cubic-bezier)
- Usar solo `transform` y `opacity` cuando sea posible
- Evitar acumulacion de efectos en el mismo trigger
